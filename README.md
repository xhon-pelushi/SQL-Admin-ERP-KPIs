# SQL Administration for ERP - KPIs and Reporting

SQL administration project for SAP Business One and Beas Manufacturing ERP systems, focusing on stored procedures, ETL processes, and business intelligence integration.

## Overview

This project demonstrates SQL administration skills for ERP systems including:
- Stored procedures for SAP Business One and Beas Manufacturing
- ETL scripts for ERP to Power BI/Tableau integration
- Backup and restore policies
- User permission management
- On-Time Delivery (OTD) reporting

## Database Management

### Stored Procedures
- **Sales Reporting**: Sales KPIs, revenue analysis
- **Inventory Management**: Stock levels, reorder points
- **Production Metrics**: Manufacturing efficiency, cycle times
- **On-Time Delivery**: OTD calculations and reporting

### Security
- Role-based access control (RBAC)
- User permission templates
- Audit logging procedures

## ETL Processes

### Data Extraction
- SAP Business One data extraction
- Beas Manufacturing data extraction
- Scheduled data synchronization

### Transformation
- Data cleansing and normalization
- Calculated fields and metrics
- Time-based aggregations

### Loading
- Power BI data models
- Tableau data sources
- SQL Server staging tables

## Backup and Recovery

- Automated backup schedules
- Point-in-time recovery procedures
- Disaster recovery documentation

## Requirements

- Microsoft SQL Server 2019+
- SAP Business One (optional)
- Beas Manufacturing (optional)
- Power BI / Tableau (for reporting)
